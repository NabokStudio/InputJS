* To stop ignore key(s) use `StopIgnoreKeys` instead of `UnIgnoreKeys`.
* Control mouse contextmenu using `GetCtxMenu`, `GetCtxUp`, `GetCtxMenuDown`, `GetCtxMenuUp`.
